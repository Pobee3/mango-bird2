# Mango Bird Independent Action Frames

Use `assets/终图_绿翅.png` as the sole character reference. Generate each frame as an independent transparent-background PNG. Do not make a spritesheet, grid, contact sheet, atlas, labels, frame numbers, shadows, scenery, branch, or cursor graphic.

Keep every frame consistent with the reference bird: lovebird/parrotlet shape, orange-red face, yellow beak, black eye with white ring, green body, green wing, blue-green tail underside, soft textured illustration/sticker style, white rim/clean cutout edge, same proportions and personality.

Canvas guidance for every frame: single whole-body bird centered with generous padding, transparent background, no text, no watermark, no extra objects. The bird should face right unless the file says otherwise. Feet should be visible when running. Wing motion should be visible through real pose changes, not motion blur.
